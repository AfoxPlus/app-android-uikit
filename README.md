# Welcome to module-android-uikit!

UIKit is an library for UI Orders application

## Setup

Run the following git commands 

```bash
git submodule init
git submodule update
```

## Usage

```kotlin
dependencies {
    ...
    implementation("com.afoxplus.android:uikit:1.1.0")
}
```
